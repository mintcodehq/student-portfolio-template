# Student Portfolio Template (Demo)

A modern, dark-themed student portfolio built with **Next.js**, **Tailwind CSS**, and **Framer Motion**.

This repository contains the **demo version** of the portfolio used for preview and deployment.

---

## Features

- Dark, minimal UI
- Fully responsive (mobile-friendly)
- Animated hero section
- Projects with images & external links
- Education section
- Resume PDF support
- Clean and easy-to-edit structure
